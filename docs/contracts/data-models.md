# 数据模型说明

当前数据模型的主版本仍在项目根目录：

- [data-models.md](/E:/OrderEz/data-models.md)

这样保留的原因是：

1. `data-models.md` 当前已经是三端共享的总入口文档
2. 根目录版本已经被现有架构文档、契约文档和后续任务说明广泛引用
3. 当前阶段先以“单一主版本”优先，避免在 `docs` 内外维护两份正文

## 当前约定
- 根目录 `data-models.md` 是主版本
- `docs/contracts/data-models.md` 只是文档目录内的导航说明页，不是第二份正文
- `docs` 内部如果需要引用数据模型，默认仍指向根目录主版本

后续如果共享文档独立迁移到单独仓库，再统一决定是否把正文迁入 `docs/contracts/` 并调整所有引用路径。
